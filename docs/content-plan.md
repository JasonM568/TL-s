## 🎯 企業貸款／企業融資 SEO 策略（Pillar–Cluster）

「企業貸款」「企業融資」是**高搜尋量的 head keyword**，競爭激烈、很難單頁硬拚。策略是用
**支柱頁（pillar）＋ 群集文章（cluster）** 建立主題權威，並把流量導到你利潤更好的支票服務。

### 三層漏斗設計

1. **入口（廣、量大）**：用「企業貸款/企業融資」相關內容吸引「有資金需求但還沒鎖定方式」的企業主。
2. **支柱頁**：`/qi-ye-dai-kuan` 是這兩個大字的主戰頁 —— 涵蓋所有融資管道總覽，H1/title/首段都佈局
   「企業貸款・企業融資」，並收攏 cluster 文章的內部連結。
3. **轉化（導流變現）**：pillar 與文章都設「手上有支票嗎？」交叉區塊，把 check-holder 導到
   **支票貼現／支票貸款**（審核快、當日撥款、你利潤較好）。

### 為什麼能贏 head keyword
- 單一頁面很難排上「企業貸款」；但**一個 pillar + 8~10 篇環繞它的長尾文章互相連結**，
  能讓 Google 認定你是「企業融資」主題的權威，連帶把 head keyword 的排名帶起來。
- 長尾文章（如「公司沒有不動產怎麼貸款」）競爭低、好排，且**搜這些的人購買意圖明確**。

### 內部連結規則（重要）
- 每篇 cluster 文章 → 至少 1 條連到 pillar `/qi-ye-dai-kuan`（錨點用「企業貸款」「企業融資」）
- pillar `/qi-ye-dai-kuan` → 交叉連到 `/zhi-piao-tie-xian`、`/zhi-piao-dai-kuan`（已內建）
- 有支票情境的文章 → 直接連支票兩頁

## 每篇撰稿規格（SEO checklist）

- 字數：1,000–1,800 字，H2 分段清楚
- 主關鍵字出現在：`<title>`、H1、首段、至少一個 H2、meta description
- 內部連結：至少 1 條指向對應金錢頁（/zhi-piao-tie-xian 或 /zhi-piao-dai-kuan）、1 條指向相關文章
- 結尾 CTA 導向 /contact
- 於 `src/lib/articles.ts` 的 `articles` 陣列新增一筆物件即可（模板已內建結構化資料、canonical、sitemap 自動收錄）
- 完成後把本檔對應列的 ⬜ 改為 ✅
